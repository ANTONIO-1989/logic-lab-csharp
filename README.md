# Logic Lab: Smart Load Balancer

### [English]
A lightweight C# console utility designed to demonstrate fundamental programming logic and resource optimization.
- **Dynamic Input**: Handles user-defined capacity and item weights.
- **Validation**: Includes real-time checks to prevent overloading.
- **Cross-Platform**: Developed on **Linux** using **.NET 8**.

---

### [Italiano]
Una utility C# da console progettata per dimostrare la logica di programmazione fondamentale e l'ottimizzazione delle risorse.
- **Input Dinamico**: Gestisce capacità e pesi definiti dall'utente.
- **Validazione**: Include controlli in tempo reale per prevenire il sovraccarico.
- **Cross-Platform**: Sviluppato su **Linux** con **.NET 8**.

---
*Developed as a technical coding challenge for C# and .NET 8 learning.*
